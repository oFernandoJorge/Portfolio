# 🚀 Portfólio Pessoal - Fernando Jorge

Um portfólio moderno e responsivo desenvolvido com Next.js, TypeScript e Tailwind CSS, apresentando minhas habilidades como desenvolvedor back-end especializado em Java.


## ✨ Características

- 🎨 **Design Moderno**: Interface limpa e profissional
- 🌓 **Tema Claro/Escuro**: Toggle entre temas com transições suaves
- 📱 **Totalmente Responsivo**: Funciona perfeitamente em todos os dispositivos
- ⚡ **Performance Otimizada**: Carregamento rápido e navegação fluida
- 🎯 **Navegação Suave**: Scroll animado entre seções
- 📧 **Formulário de Contato**: Modal integrado para envio de emails
- 🎭 **Scrollbar Personalizado**: Design minimalista e elegante

## 🛠️ Tecnologias Utilizadas

### Frontend
- **Next.js 14** - Framework React para produção
- **TypeScript** - Tipagem estática para JavaScript
- **Tailwind CSS** - Framework CSS utilitário
- **Shadcn/ui** - Componentes UI modernos
- **Lucide React** - Ícones SVG otimizados

### Funcionalidades
- **React Hooks** - useState, useEffect para gerenciamento de estado
- **CSS Animations** - Transições e animações suaves
- **Responsive Design** - Mobile-first approach
- **Dark Mode** - Sistema de temas dinâmico

## 🚀 Como Executar

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação

1. **Clone o repositório**
\`\`\`bash
git clone https://github.com/FJdevdev/portfolio-dev.git
cd portfolio-dev
\`\`\`

2. **Instale as dependências**
\`\`\`bash
npm install
# ou
yarn install
\`\`\`

3. **Execute o projeto**
\`\`\`bash
npm run dev
# ou
yarn dev
\`\`\`

4. **Acesse no navegador**
\`\`\`
http://localhost:3000
\`\`\`

## 🎨 Personalização

### Cores
As cores principais podem ser ajustadas no arquivo \`tailwind.config.ts\`:

\`\`\`typescript
colors: {
  primary: "221.2 83.2% 53.3%",    // Azul principal
  secondary: "210 40% 96%",         // Cinza claro
  // ... outras cores
}
\`\`\`

### Conteúdo
Para personalizar o conteúdo, edite as variáveis no arquivo \`app/page.tsx\`:

\`\`\`typescript
const skills = {
  backend: ["Java", "Spring Boot", "C#", ".NET"],
  frontend: ["HTML", "CSS", "JavaScript", "React"],
  // ... suas habilidades
}
\`\`\`

## 🚀 Deploy

### Vercel (Recomendado)
1. Conecte seu repositório GitHub à Vercel
2. Configure as variáveis de ambiente (se necessário)
3. Deploy automático a cada push

### Netlify
1. Conecte seu repositório
2. Configure build command: \`npm run build\`
3. Publish directory: \`out\`

### GitHub Pages
1. Configure GitHub Actions para build
2. Use \`next export\` para gerar arquivos estáticos
